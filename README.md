# Homebrew repository for `bcvk`

This is packaging for [`bcvk`](https://github.com/bootc-dev/bcvk) on [Homebrew](https://brew.sh) for Linux as this is the only supported platform by that project currently

## How do I get these

```bash
brew install bootc-dev/bcvk/bcvk
```

You can also download the latest dev version by running this instead:

```bash
brew install --head bootc-dev/bcvk/bcvk
```
